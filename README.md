# Système de Gestion Portuaire (DBA Project)

Ce projet implémente l'architecture base de données pour la gestion d'un port, incluant la gestion des quais, des opérations, des ressources humaines et de la facturation.

## Contenu

- **Modélisation** : Scripts DDL complets.
- **Sécurité** : Gestion fine des droits via des rôles (Manager, RH, Finance, etc.).
- **Données** : Jeu de données de test (DML).
- **Documentation** : Rapport complet disponible dans le dossier 'docs/'.
